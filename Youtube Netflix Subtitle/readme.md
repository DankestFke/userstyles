<h1 align="center"> YouTube Netflix Subtitle </h1>
Derives from @ang3lo_azedo's "YouTube Netflix Subtitles". This userstyle focused on customization on subtitles.

## Features
- Subtitle Customization
  - Normal / All subtitle toggle
    
- Subtitle font
  - Colour
  - Italic
  - Weight
  - Font type
  - Align
  - Transformation
  - Outline
    - Color
    - Thickness
      
- Subtitle window
  - Centered
  - Bottom offset
